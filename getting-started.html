Here's how to get up and running with Kindling.

Quick Start

1

Install Kindling

Open the downloaded file and follow the installation prompts.

macOS Security Note

If you see "developer cannot be verified," go to System Settings → Privacy & Security and click Open Anyway. Full guide →

Windows SmartScreen Note

If Windows shows "protected your PC," click More info then Run anyway. Full guide →

2

Import Your Outline

Choose Import from the start screen. Kindling supports Plottr, Markdown, yWriter, and Obsidian Longform projects.

3

Start Writing

Your scene beats appear in a side panel while you draft. No more switching between apps!
