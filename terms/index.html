2. Description of Software

Kindling is a free, open-source desktop application for fiction writers. It runs locally on your computer (macOS, Windows, Linux). It imports outline files (Plottr .pltr, yWriter .yw7, Markdown .md) and provides a scaffolded writing environment. It exports to DOCX and Markdown formats. Kindling does not require an account, internet connection, or subscription.
