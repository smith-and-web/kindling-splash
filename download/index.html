First-time setup

macOS: Kindling is not yet signed with an Apple Developer certificate. On first launch, right-click the app and select "Open", then click "Open" in the dialog. You only need to do this once.

Windows: Windows SmartScreen may show a warning for unrecognized apps. Click "More info" → "Run anyway". This is because Kindling is new and building its reputation with Microsoft.

This is common for new indie software. Nothing to worry about.

How to uninstall Kindling

Kindling can be completely removed from your computer at any time. Your project files are not deleted when you uninstall Kindling — they remain wherever you saved them.

macOS

  1. Open Finder and go to the Applications folder.
  2. Drag Kindling to the Trash (or right-click → Move to Trash).
  3. Empty the Trash.

Kindling does not install any background services, launch agents, or system extensions. To remove application data (preferences, not project files), delete ~/Library/Application Support/com.kindlingwriter.app/ if present.

Windows

  1. Open Settings → Apps → Installed apps (or Apps & features on Windows 10).
  2. Search for Kindling.
  3. Click Uninstall and follow the prompts.

Alternatively, use Control Panel: Programs and Features → select Kindling → Uninstall. Kindling does not install background services or startup items. To remove application data: delete %APPDATA%\com.kindlingwriter.app\ if present.

Linux (AppImage)

  1. Delete the Kindling AppImage file (wherever you saved it).
  2. If you created a desktop entry, remove ~/.local/share/applications/kindling.desktop.

To remove application data: delete ~/.config/com.kindlingwriter.app/ or ~/.local/share/com.kindlingwriter.app/ if present.
